# Quickstart: Testing PWA Features

1. Run frontend: `cd frontend && npm run dev`
2. Open in Chrome Android:
   - Verify "Add to Home Screen" banner appears
   - Install and verify icon/splash screen
3. Open in Safari iOS:
   - Use Share > Add to Home Screen
   - Verify fullscreen mode and status bar color
4. Verify favicon in desktop browsers
5. Check manifest.json: http://localhost:5173/manifest.json
